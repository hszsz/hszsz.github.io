Sql server2017安装教程

一.国际惯例先介绍

  SQL是英文Structured Query Language的缩写，意思为结构化查询语言。SQL语言的主要功能就是同各种数据库建立联系，进行沟通。按照ANSI(美国国家标准协会)的规定，SQL被作为关系型数据库管理系统的标准语言。SQL Server是由Microsoft开发和推广的关系数据库管理系统（RDBMS）。

二.安装

1.还是先下载安装包，安装包的下载地址我会贴在最底下

2.点击SQLServer2017-x64-CHS.iso找到setup.exe点击打开

3.找到图片中红色标记处安装

4.点击第一个全新安装

5.出现这个界面选择指定可用版本

6.然后按照标记操作


7.到了这里会显示Windows防火墙，打开电脑控制面板找到防火墙设置



8.在这里关闭防火墙然后点击确定

9.回到sql server安装界面点击重新运行

10.出现下面这个全部已通过点击下一步

11.这里选择全选，可以修改安装位置。

12.然后找到这两个把它不选中


13.在这里选择默认实列点击下一步

14.这个默认下一步

15.找到这个选中后点击下一步

16.到了这一步点击红色标记处依次设置

17.选择表格模式，添加当前用户，然后点击下一步

18.这里选择新建SSL证书

19.这里直接下一步

20.这里添加当前用户然后下一步

21.这里直接下一步

22.前面设置好了，到了这里直接点击安装


23.到了这里算是安装完成了

24.这里安装sql UI找到这个文件直接点击

25.直接点击安装，安装时间可能有点长耐心等待


26.到了这里图形化界面算是安装成功了

27.点击开始找到扳手锤交叉库应用程序打开数据库，连接数据库

28.到了这里注意红色标记的地方鼠标点击一下右键启动打开服务。

到这里整个SQL Server2017 安装成功。
